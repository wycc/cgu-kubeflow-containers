jupyterlab-ml notebook 使用說明

創建並啟動notebook

  • 創建notebook時，請選擇jupyterlab-ml的image
  • 如果要進行機器學習運算，記憶體需要足夠大，建議4G以上
  • 如果要使用GPU加速，請加入至少一張NVIDIA GPU

使用notebook

先上傳py檔案或ipynb檔案，打開檔案，然後選擇python kernel

依據使用的深度學習框架，選擇對應的環境

選擇Run -> Run All Cells，執行整份程式

如何自行安裝套件

可以在儲存格中使用pip或conda安裝套件

如何使用R語言，或是終端機 Terminal

開啟新視窗，並選擇R語言或終端機來啟用
